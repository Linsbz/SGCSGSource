# Sistema de Gerenciamento de Conteúdo para Sites Genéricos - SGCSG
## Cadastros:
```
- usuário (tipos: comum, redator, administrador)
- notícia
- anúncio
- área de anúncio
- enquete e opção de voto
- formulário e campo de formulário
- tema
```
## Processo de negócio:
```
- postagem de anúncio
- postagem de notícia
- voto
- resposta de formulário
- compra de tema
```
